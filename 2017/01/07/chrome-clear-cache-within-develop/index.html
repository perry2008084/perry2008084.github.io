Chrome浏览器下清除缓存的方法总结

发表于 2017-01-07 更新于 2017-01-11 分类于 前端开发

开发过程中，修改js和css，希望浏览器能够加载最新的代码，需要清除缓存，记录几种Chrome下的方法。

Chrome下清除浏览器缓存刷新页面的方法

开发者工具中禁用缓存选项

打开Chrome的开发者工具，Network下有个Disable cache选项，勾上即可。

注意点： 该选项只有在开发者工具保持打开状态有效。

浏览器插件：Cache Killer

安装浏览器插件，可以打开或者关闭。

右键点击刷新按钮

右键点击刷新按钮，可出现三个选项：

  • 正常重新加载
  • 硬性重新加载
  • 清除缓存并重新加载

可选择“清除缓存并重新加载”，来清除全部缓存。
