构造函数模式

构造函数模式，通过创建自定义的构造函数，定义自定义对象类型的属性和方法。

原型模式

通过函数的prototype属性进行所有实例共享来添加属性和方法。

优点：

组合使用，构造函数模式用于定义实例属性，原型模式用于定义方法和共享的属性。
