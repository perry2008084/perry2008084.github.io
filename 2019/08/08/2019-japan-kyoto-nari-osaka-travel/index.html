2019日本京都奈良大阪自由行游记

发表于 2019-08-08 更新于 2019-08-09 分类于 生活

本次旅游了日本的京都、奈良、大阪三个地方，共花了8天7夜的时间，对本次的旅行记录一下回顾和感想，其中分为三个部分来描述，游记、攻略和感想。

游记

关于这次的旅行，是年初的时候就计划好的，关于地点是台北和日本京都选择其一的，最终选择了日本，理由是先感受一下日本的文化和风情；关于时间，起初计划的时间是9月份的，正好这段时间离职了，老婆也正好也有时间，就排上这次旅行。

总的图片集合可以访问日本自由行图片集合

接下来是关于本次走过的几个地方的照片和描述：

京都

Day1

第一站是从关西国际机场坐JR haruka（类似城际列车）到京都，到达时间是晚上7点多，日本和中国北京时间的时差是快一个小时。酒店办理入住后，就出门浪一波，找些吃的。

接着第二天打卡的当然是人气很高的伏见稻荷大社，供奉的是日本掌握五谷丰收的稻荷，这里很出名的就是千本鸟居，旁边有一条街是小吃街，有很多好吃的东西。

逛完稻荷大社，下午去了锦市场，食材一条街，说实话，来这里要赶早，因为他们基本上5点多就开始陆续关门了。在这里，可以买到很多做菜的食材，也有一些是熟食可以现场吃，总之，可以逛逛顺便吃吃。

逛完锦市场，准备去祗园的路上，正好赶上了祗园祭，八坂神社门前的那条路两边围满了人，路上有交警维持秩序，然后就是祭祀的人员一路过来，各种服装打扮的。

看了一段时间的活动，我们就沿着路线往祗园的花见小路走了，途中会路过鸭川，坐在鸭川旁的石墩上，看着傍晚的鸭川，心情很舒畅。旁边有很多年轻人，有的骑个自行车过来，将自行车放在一旁看书，或躺着思考，或看着河流。

看完鸭川，继续前行，走到了祗园的花见小路，据说这里可以看到艺伎，其实看到穿着和服的都是游客，有中国的，也有欧美的，话说，来京都游玩的欧美人真是多啊。带着闲适的心情，走了花见小路、白川，然后找了个饭店吃了个饭，等待夜晚降临去看八坂神社的夜景。

八坂神社本身并不大，参观完神社，我们就在神社门口等待祗园祭的活动人员归来。大约晚上9点多，大队伍就回来了。

看完祗园祭，差不多晚上10点左右。

Day2

我们订了两晚的酒店，然后换到民宿去了，民宿的一大好处就是可以做饭，也能体验一下当地的日常生活起居。这一天的行程和做攻略的计划有一些不同，做了调整，先去了京都国际漫画博物馆，本来打算去买一套在其他书店比较难买到的《虫师》，但是去了之后才知道，里边卖漫画书的仅仅只是一个小书店，这个书店还大部分是卖纪念品和文具的。主要就是进去逛逛日本漫画的历史，以及一些漫画作家的作品展。

这是大数据的统计，日本销售前三的动漫作品

这是日本一个漫画作家的书房

逛完漫画博物馆，我们就顺便到二条城了，因为两者相距也就10分钟的步行路程。这两个地方同样都是需要买票的，不过我们来的是大夏天的中午时分，逛二条城是有点艰难的，有很多建筑是室外的，即使是室内的建筑，走进去也是十分炎热的。二条城的维基百科介绍，是江户幕府第一代征夷大将军德川家康作为守护京都御所以及到京都拜见天皇住宿的地方。这里没有拍什么照片，主要是太热了。

日本的酒店和民宿的入住和退房时间，下午3点和早上10点，相对国内的时间而言，比较严格，不过酒店不管入住还是退房，是可以寄存行李的。我们逛完二条城，就回酒店拿行李准备到民宿办理入住，鉴于午后这段时间很热，就在民宿里吹空调了，到傍晚时分再出来活动。

夜晚出来，拍到的京都塔夜景。

Day4

这一天是雨天，不过没有阻挡我们的出行，去的是岚山，去看竹林。这里有个小插曲，我们是坐的公交车去的，原先在谷歌地图上搜的，看到岚山的山上有标注竹林，然后我们的公交就坐到那附近的一个站点下车，直接去上山的入口，到了入口一看，一个牌子写着，这个山是封锁的，为了给后代保存的，不让非相关人员进入，这种事情还真得经历了才知道。

然后我们就往渡月桥和龟山公园前进了，竹林是在龟山公园中，虽然是下雨天，但是游客依然没有少。

奈良

Day3

接下来的这一天，是在奈良度过的，从京都做一个JR奈良线很方便，主要是逛了奈良公园，公园里很大，景点有多个，包括东大寺、春日大社及大大小小的神社。

奈良给我的最大印象是，这里的树木都很粗，年纪应该都很大，而且走在奈良公园的树林里，给人一种很宁静的感觉，有点类似日本动漫《龙猫》里的那种夏日的宁静。

沿着东大寺的后山的路线前进，可以路过若草山，若草山旁边的那一条真的很漂亮，蓝天、白云、草坪，给人一种心旷神怡的感觉，如果春秋来这里，可能会更舒服。

奈良公园里的鹿确实是多，就在马路边，也不害怕游客。

回来的过程中，发生了一件小事情，是关于行李托运的问题。我们买了不少化妆品，是直接拿在手上而没有办托运，导致在过安检的时候，有几样东西无法通过安检，没办法只能去柜台重新办理托运，导致时间耗费了不少，而办理机票手续的排队时间很长，差点延误了登机时间。

攻略

如果是自由行，攻略是少不了的，不像跟团游，到了有导游指引，自己走，没有规划可能就是一脸懵了。

行程攻略

这是我们这次的行程攻略：

主要是要提前确定想要游玩的景点和地方，这样去了之后就有个方向。当然这个规划的行程和实际的行程或许会有出入，根据最终的心情和天气等决定。

交通

我们这次去的关西地区，提前查了那边通用的公交卡是Icoca，所以提前在淘宝上买了两张，在坐地铁和公交的时候确实方便不少。这个公交卡还可以在一些便利店和商场购物使用。

这里有一个要注意的是，从关西机场到京都的JR haruka凭Icoca可以购买优惠票，每张票可以优惠大约1000多日元。

网络

这里涉及两个方面，一个是需要提前租一个WiFi，方便落地后直接上网；另一方面，我们是在联通网上营业厅开通手机的国际漫游业务，这样到了日本就可以开启漫游功能，可以免费接收短信，打电话的资费是比较贵的，可以通过网上了解。

现金

通过提前了解，可以估算每天的花销，根据这个花销提前在国内换一些日元的现金，当地的购物有很多都是现金的，不像国内微信和支付宝，不过那边有很多商场是支持微信和支付宝的。

感想

这里感触比较深的有几点:

  1. 大部分人的态度都比较友好的，尤其是在购物和大街上。
  2. 汽车礼让行人的习惯上。
  3. 马路上的卫生情况，大部分的情况都是比较干净的，要吐槽的是，大街上找垃圾桶比较痛苦。
  4. 关于文化的传承，像祭祀这样的传统能代代相传，确实值得让人敬佩。
